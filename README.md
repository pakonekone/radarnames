# RadarNames

Domain portfolio management and outbound sales pipeline for SBV domains.

## Overview

RadarNames is the operational hub for sourcing, evaluating, and selling premium domains. The portfolio currently holds **27 domains** acquired primarily through Dynadot closeout auctions at ~€14 each.

## Strategy

- **Outbound sales**: Direct outreach to exact brand-match buyers (lawbotica, ashata, adresshotels, hotelvireo, alcatelontouch)
- **Marketplace passive**: Listings on Afternic, Sedo, and Atom with LTO enabled
- **Development**: Select domains (coffeegare.com, estaturas.com) developed as niche sites
- **Scanning**: Daily closeout scanning via Dynadot API for new opportunities
